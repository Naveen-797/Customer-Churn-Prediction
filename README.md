# Customer Churn Prediction

## Objective
Analyze telecom customer data to identify key churn drivers and build a machine learning model to predict customer churn.

## Dataset
Telco Customer Churn Dataset (IBM sample dataset)

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Key Insights
- Month-to-month contracts show highest churn (~42%)
- Low tenure customers are high risk
- Higher monthly charges increase churn probability
- Long-term contracts reduce churn significantly

## Model Performance
Balanced Logistic Regression:
- Accuracy: ~76%
- Recall (Churn): ~82%

## Business Recommendations
- Encourage long-term contracts
- Target low-tenure customers for retention
- Provide incentives to high-charge customers
